# Titanic 生还率分析

基于 Titanic 乘客数据，使用 Pandas 进行探索性数据分析，挖掘影响生还率的关键因素。

## 分析内容
- 数据清洗：处理缺失值（Age 中位数填充，Cabin 列删除，Embarked 缺失行删除）
- 性别对生还率的影响
- 舱位等级对生还率的影响
- 性别与舱位的交叉分析
- 年龄分组对生还率的影响

## 主要发现
- 女性生还率 74.0%，男性仅 18.9%
- 一等舱生还率 62.6%，三等舱仅 24.2%
- 一等舱女性生还率高达 96.7%，三等舱男性仅 13.5%
- 儿童生还率最高（57.9%），老年人最低（19.0%）
